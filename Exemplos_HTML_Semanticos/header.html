Elemento Header

O header elemento representa um contêiner para conteúdo introdutório ou um conjunto de links de navegação. Um header elemento normalmente contém:

  • Um ou mais elementos de título
  • Logotipo ou ícone
  • Informações de autoria

Nota: Você pode ter vários header elementos em um documento HTML. No entanto, header não pode ser colocado dentro de um footer ou address outro header elemento.
